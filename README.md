## Product Overview

MiniHexa is an educational hexapod robot developed by Hiwonder. It features a compact design with six legs, enabling flexible and diverse movements. The robot supports multiple programming methods including Python (MicroPython), Arduino, Scratch, and APP remote control, making it perfect for learning robotics, programming, and AI applications.

MiniHexa is equipped with AI capabilities including computer vision, voice interaction, and large language model integration. It provides a comprehensive platform for students and hobbyists to explore robotics, artificial intelligence, and embodied intelligence applications.

## Hardware Configuration
- **Controller**: ESP32-based microcontroller
- **Servos**: 6 high-torque servos for leg control
- **Sensors**: Camera, microphone, distance sensors
- **Communication**: WiFi, Bluetooth, UART serial
- **Power**: Rechargeable battery system
- **Expansion**: GPIO pins for additional sensors and modules

## Getting Started

### For Python Programming
1. Set up MicroPython development environment
2. Follow tutorials in `python-programming-projects/`
3. Start with basic motion control
4. Explore AI vision and voice projects

### For Arduino Programming
1. Install Arduino IDE
2. Open projects in `arduino-programming-projects/`
3. Upload code to MiniHexa
4. Test and modify examples

### For Scratch Programming
1. Install Scratch software
2. Load projects from `scratch-programming-projects/`
3. Create visual programs with drag-and-drop blocks

### For APP Control
1. Install MiniHexa mobile app
2. Connect to robot via WiFi/Bluetooth
3. Use remote control interface
